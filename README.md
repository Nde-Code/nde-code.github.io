# 🌐 nde-code.github.io:

Code source de mon site web personnel.

Visitez le site ici: [nde-code.github.io](https://nde-code.github.io/)

Prenez une seconde pour voir le code dans Visual Studio Code: [https://github1s.com/Nde-Code/nde-code.github.io](https://github1s.com/Nde-Code/nde-code.github.io)

## 📚 Librairies utilisées:

- [Font Awesome](https://fontawesome.com/)

- [jQuery](https://jquery.com/)

- [Math.js](https://mathjs.org/)

- [Plotly.js](https://plotly.com/javascript/)

- [Ace.js](https://ace.c9.io/)

- [Leaflet.js](https://leafletjs.com/)

## 🛠️ Outils et ressources:

- [Visual Studio Code](https://code.visualstudio.com/)

- [Pixlr](https://pixlr.com/fr/)

- [Logomakr](https://logomakr.com/)

- [Perchance – AI Icon Generator](https://perchance.org/ai-icon-generator)

- [Flaticon](https://www.flaticon.com/): pour les diverses icônes utilisées  

- [Pixabay](https://pixabay.com/fr/illustrations/bleu-triangles-polygone-contexte-1761175/): pour l’image de fond.

## 🧩 Services tiers:

- [Goatcounter](https://www.goatcounter.com/): outil de statistiques web respectueux de la vie privée.

- [MapTiler](https://www.maptiler.com/): service de tuiles "raster" pour les cartes Leaflet.js.

## 🎯 Notes:

La base de données intégrée à mon **logiciel de désintégration radioactive** a été conçue **entièrement par mes soins**, à partir de sources variées.  

Ce travail a nécessité un investissement conséquent en temps et en recherche.

Deux versions sont disponibles :

- une **version lisible** et facilement modifiable [ici](https://gist.github.com/Nde-Code/0dc819d80ee1d593d7d27d138fc68292). 

    - Toutes les demandes de modifications, commentaires, avis, etc..., doivent être faites sur ce Gist. 

- une **version optimisée et minimisée** pour un usage en ligne performant.

    - Version très simplement minimisée via un outil, très peu pratique pour la maintenance.

> 🔓 Son utilisation est libre, mais une **mention de crédit** serait grandement appréciée.
