Notes

Les données sur les météorites tombées sur Terre proviennent de la [NASA].

Le but de ce projet est de réaliser des statistiques basées sur l'aléatoire (générale ou par zone/continent), des météorites tombées sur Terre. Pour des recherches plus avancées, vous pouvez consulter la documentation de l'API [Ici].

Les données sont hébergées dans une base de données Google (Firebase) et récupérées via une API (programmée par mes soins), hébergée sur Deno Deploy.

J'utilise une tuile fournie par [MapTiler] pour avoir un fond de carte plus sympa qu'OpenStreetMap.

Pour prévenir les abus, l'API derrière le logiciel utilise votre adresse IP, mais celle-ci est hachée (via SHA-256) avec un "SALT" solide et robuste. Les données sont conservées au maximum pendant 24 heures afin de respecter au mieux le RGPD. Aucune IP brute ni aucune autre information personnelle n'est collectée. N'hésitez pas à me contacter pour toute question relative à ce sujet. En utilisant ce logiciel ou/et l'API associée, vous confirmez être en accord avec les présentes conditions.
